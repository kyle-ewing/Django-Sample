# Django-Sample

Simple django app to showcase ability to create a basic CRUD app. App contains validations for user and trip creation. Passwords are successfully hashed using bcrypt on creation of user.

### How to run
1. Set up virtual environments and include the following packages:
* bcrypt
* Django

2. In terminal run: python manage.py migrate
3. In terminal run: python manage.py runserver
4. Navigate browser to localhost:8000
